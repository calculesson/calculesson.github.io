2nd grade

The calculation you have to do increases and becomes more difficult.

What You Can Learn Here

  • Calculating Expressions

    By expanding on what you learned in elementary and first grade, you will master letter formulas.
    When Do You Use It?
    When you try to make expressions, you may create complicated one.
    Also, there will be more units that require a lot of calculations from this grade level.
    The calculation skills acquired in this unit will be useful in daily life and other units.

  • Simultaneous Linear Equations

    It means an equation containing two variables. When Do You Use It?
    There is not always one number you don't know but want to know.
    If you have the ability to create and solve equations, you can know what number goes into a variable.

  • Linear Functions

    A linear function means that y is a function of x and is represented by a linear expression of x.
    In this unit, you also learn about graphs using linear functions.
    When Do You Use It?
    You will be able to visualize and organize linear expressions as graphs.
    Also, you can draw most straight lines on the coordinate plane.
    It will also be useful for making predictions about data.

  • Utilizing Data

    You will use various values to find the scatter and other features of the data.
    When Do You Use It?
    When you do a survey and analyze its results, you will be able to use the knowledge gained in this unit to read what the data can tell you.
    If you use the data effectively, your explanations will be more persuasive.
